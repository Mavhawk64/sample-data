# sample-data

Simple repository to GET sample data for testing parsers / cURL requests.

## Available sample files

- `/users.json`
- `/users.csv`
- `/users.tsv`
- `/users.xml`
- `/users.html`
- `/users.graphql`
- `/users.uql`
- `/users.groq`

All of the files use the same three-user sample dataset where the format allows it. The GraphQL, UQL, and GROQ files provide matching query examples that can be used with the JSON dataset.
